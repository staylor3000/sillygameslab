Building in the open

Dev Log — making it up
as we go.

This is a blog composed of short dev logs from building browser games and small projects with Claude and Claude Code (including the blog itself), with the aim of "building in the open."

I'm a self-taught data engineer by trade, not a web developer or game designer, so this is very much a steep learning curve documented for the world to see.

No long articles. Just honest bullet points about what got built, maybe what didn't, and what I learnt along the way. After all, it's the journey not the destination that matters, apparently?
