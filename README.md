# Silly Games Lab

Landing page for [sillygameslab.com](https://sillygameslab.com) — an indie game studio making simple, joyful web games.

## Stack

Pure HTML/CSS — no build step. Open `index.html` directly or serve locally:

```
python3 -m http.server 8080
```
